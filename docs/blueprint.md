# **App Name**: Luni Site

## Core Features:

- Simulated Authentication: Simulate user authentication via AuthContext with login, register, and forgot password pages.
- Main Layout: Implement a main layout with a fixed header and conditional navigation (bottom nav for mobile/tablet, header nav for PC).
- Profile Page: Design a profile page displaying user info, selected exam type, study time, exam simulator, and study time distribution chart.
- Content Sections: Implement content sections (videos, content, quizzes) with search and filtering capabilities, displaying content in a grid of cards.
- Recommended Videos: A tool powered by AI that recommends videos from available educational content on the internet
- Simulated Exam Flow: Create a simulated exam flow with a question display, timer, navigation buttons, and submission simulation.
- Updates Page: Develop a "Coming Soon" page for the updates section.

## Style Guidelines:

- Primary color: Oxford Blue (#325A73) to provide a professional and serious, yet trustworthy tone to the e-learning platform. Oxford blue is used for the primary interactive elements like buttons and active states.
- Background color: Light Gray (#F0F0F0) to ensure a clean and neutral backdrop that highlights content without distraction. This background will support legibility and reduce eye strain during prolonged use.
- Accent color: Teal (#008080) as an accent for highlights and calls to action. This color will attract user attention to important actions without overpowering the design.
- Headline font: 'Roboto' (sans-serif) for headers, offering a modern and easily readable style that works well on digital platforms.
- Body font: 'Open Sans' (sans-serif) for body text, chosen for its clarity and legibility across different screen sizes, making it suitable for the detailed content expected in an e-learning environment.
- Use icons from Lucide-react for a consistent and clean interface, focusing on simple and recognizable symbols that aid in navigation and feature identification.
- Employ a modern, clean, and minimalist layout with rounded corners and subtle shadows to add depth to components, enhancing the visual appeal and user experience.
